# Train-Booking-System
This project was created using Java and JavaFx
This is a menu driven program for booking seats on a train with 42 seats in the A/C 
Compartment in Denuwara Manike train to Badulla.

This program is for the use of the passenger to mainly booked train seat for the specified 
root and date. 
And they can delete booking records.

Can use this application for :
      1. User can book train seat
      2. User can delete booking record
      3. Can view all seats (Display booked seats and empty seats separately)
      4. Can view all empty seats
